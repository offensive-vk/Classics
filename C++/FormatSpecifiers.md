| Format Specifier | Description                                       |
|------------------|---------------------------------------------------|
| %d or %i        | int integer                                       |
| %f              | float single-precision decimal type               |
| %lf             | double high precision floating point data or number |
| %c              | char character                                    |
| %s              | for strings strings                              |
| %ho %o %lo      | Octal                                             |
| %hd %d %ld      | Decimal                                           |
| %hx / %hX %x / %X %lx / %lX | Hexadecimal                             |
